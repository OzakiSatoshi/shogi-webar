# 駒面画像のクレジット

`fu.svg`〜`ryu.svg`（14種）は、以下のオープンライセンス素材を使用しています。

- 出典: **Ka-hu / shogi-pieces**（`kanji_red_wood` セット, 先手向き「0」）
  https://github.com/Ka-hu/shogi-pieces
- ライセンス: **CC BY 4.0**（表示）
  https://creativecommons.org/licenses/by/4.0/
- 作者: Ka-hu

本アプリでは、これらの標準的な駒面を AR マーカーの見た目に使用し、
追従・個体識別のために駒面へ目立たない固有の杢（もく）を薄く重ねています。

ファイル名と駒の対応：
fu=歩 / kyo=香 / kei=桂 / gin=銀 / kin=金 / kaku=角 / hi=飛 / ou=王 /
to=と / nkyo=成香 / nkei=成桂 / ngin=成銀 / uma=馬 / ryu=龍
